**Travel Companion**
**Overview**

The Travel Companion App is an Android application that allows users to easily convert values across different categories. It is designed mainly for travellers who need quick and reliable conversions while moving between countries.

**Features**

1. Currency conversion
2. Fuel and distance conversion
3. Temperature conversion
4. Input validation
5. Error handling
6. Simple and clean user interface

**Technologies Used**

Kotlin,
Android Studio,
XML for UI design.

**Conversion Categories**
1. Currency 
Supports conversion between:
USD, AUD, EUR, JPY, GBP

2. Fuel & Distance
Includes:
mpg, km/L, Gallon (US), Liters, Nautical Mile, Kilometers

4. Temperature
Includes:
Celsius, Fahrenheit, Kelvin

**Validation**
Does not allow empty input
Blocks invalid (non-numeric) values
Handles same unit conversions properly
Prevents negative values for fuel inputs

**Steps**
1. Open the project in Android Studio
2. Build the project
3. Run the app on an emulator or Android devic
